📊 Employee Turnover Prediction

This project focuses on predicting employee turnover using machine learning techniques. The dataset (employee_turnover.csv) contains various employee-related features such as job satisfaction, performance, and work conditions.

Three models were implemented and compared:

1. Logistic Regression – for baseline classification
2. Lasso Regression (L1 Regularization) – for feature selection and reducing overfitting
3. Ridge Regression (L2 Regularization) – for handling multicollinearity and improving generalization

The models were trained and evaluated to identify the most important factors influencing employee attrition and to improve prediction accuracy.
