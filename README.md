# Employee Attrition Prediction

## 📌 Overview
This project predicts whether an employee is likely to leave the organization using Machine Learning. Logistic Regression models with L1 (Lasso) and L2 (Ridge) regularization are implemented and compared to identify the best-performing model.

## 🚀 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Jupyter Notebook

## ⚙️ Workflow
- Data Preprocessing
- Train-Test Split
- Logistic Regression Model
- L1 & L2 Regularization
- Model Evaluation

## 📊 Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-Score
- Classification Report

## 📈 Results
- Logistic Regression Accuracy: **85.9%**
- Logistic Regression (L1) Accuracy: **87%**
- Logistic Regression (L2) Accuracy: **85.9%**
- Best Performing Model: **Logistic Regression (L1 regularization)**
